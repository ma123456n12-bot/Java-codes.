
# Overview
This repository contains intermediate-level Java programs focused on Object-Oriented Programming and clean coding practices. It includes multi-class projects and logic-based applications.

## Concepts Covered
- Classes and Objects  
- Inheritance and Polymorphism  
- Encapsulation and Abstraction  
- Exception Handling  
- File Handling (Basic)  
- Collections Framework  

## Technologies
- Java (JDK 8+)  
- VS Code / IntelliJ / Eclipse  

## How to Run
Compile and run using:
javac Main.java
java Main
javac Main.java
java Main
